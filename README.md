# TwinBackend

Backend API for Twin AI Avatar System.

## Tech Stack

- Node.js
- Express.js
- MySQL
- Git & GitHub

## Features

- Create Appearance
- Get All Appearances
- Update Appearance
- Delete Appearance
- Validation Middleware
- Error Handling

## API Endpoints

### Get All Appearances

GET /appearance

### Create Appearance

POST /appearance

### Update Appearance

PUT /appearance/:id

### Delete Appearance

DELETE /appearance/:id

## Run Project

```bash
npm install
node server.js
```

Server runs on:

```
http://localhost:5000
```
